# 第一卷關鍵NPC列表

## 概述

第一卷的主角年齡跨度：8→12歲。主角身處塔恩鎮及周邊，偶爾隨父親進格倫城。
這個階段的核心任務：保住父親、建立第一個班底（找到塔利）、積累初始資源。

---

## 塔恩鎮 NPCs

### 葛瑞斯（Greys）——塔恩鎮鎮長助理

| 項目 | 內容 |
|------|------|
| 年齡 | 45歲 |
| 職業 | 鎮長助理（處理實際行政事務） |
| 性格 | 務實，輕度貪腐，但從不在大事上犯糊塗 |

**對主角的意義**：
- 掌握塔恩鎮所有商業往來的通行文件
- 主角第一次「合法做生意」需要他的蓋章
- 他不是敵人，也不是真正的盟友——是一個可以用小錢打點的行政障礙

**互動節點**：
- 第1卷第3章附近：主角通過父親的名義，讓葛瑞斯幫一個小忙。葛瑞斯沒有懷疑，因為這看起來只是一個孩子幫父親跑腿。

---

### 班尼（Benny）——塔恩鎮集市的藥草收購商

| 項目 | 內容 |
|------|------|
| 年齡 | 38歲 |
| 職業 | 藥草收購中間商，轉賣給格倫城的藥商 |
| 性格 | 精於算計，對小孩態度輕蔑，但認可真正的利益 |

**對主角的意義**：
- 主角的第一個正式「交易對象」
- 主角利用前世記憶中對某種藥草的知識，向班尼提供了一個品質判斷訣竅——換來一個小額的固定收購合約
- 班尼不重要，但這次交易是主角學到「如何把知識轉化為資源」的第一課

**互動節點**：
- 第1卷第4-5章：主角幫父親在集市跑腿時，主動接觸班尼。讓班尼覺得「這孩子懂一點東西，可以當個小採購」。

---

### 老德（Old Dex）——退役的前邊境軍士兵，現在塔恩鎮打鐵

| 項目 | 內容 |
|------|------|
| 年齡 | 62歲 |
| 職業 | 鐵匠，兼修兵器 |
| 性格 | 話少，對無聊的人不耐煩，對懂行的人另眼相看 |

**對主角的意義**：
- 主角前世記憶中有「如何辨別刀的質量」的知識，老德發現這個孩子懂
- 成為主角在塔恩鎮的第一個「情報節點」——退役老兵的八卦圈子其實廣
- 後期提供一個關鍵情報：格倫城某個貴族管家在採購武器，主角從這裡推斷出一個政治動向

**互動節點**：
- 第1卷第6章附近：主角跟老德聊刀，老德意外地把他當「自己人」，開始隨口說一些平時不說的話。

---

### 莎拉嬸（Aunt Sara）——藥師之妻，梅拉的母親

| 項目 | 內容 |
|------|------|
| 年齡 | 36歲 |
| 職業 | 協助丈夫管理藥鋪，負責收帳 |
| 性格 | 熱情、話多，對鄰里關係極其重視 |

**對主角的意義**：
- 梅拉的母親，主角第一次登門拜訪就是以「來買藥」為藉口
- 莎拉嬸是「把主角和梅拉的友誼合法化」的社交節點（兩家的大人認識，孩子往來才不奇怪）
- 她對主角的評語：「那個莫爾家的孩子，說話太老成，但眼神很好，不是壞孩子。」

## 格倫城 NPCs（主角隨父親進城時接觸）

### 哈維・托恩（Harvey Torn）——格倫城貨運碼頭監工

| 項目 | 內容 |
|------|------|
| 年齡 | 44歲 |
| 職業 | 碼頭監工，管理貨運雜工的派工 |
| 性格 | 現實，不講情面，但說話算話 |

**對主角的意義**：
- 塔利在碼頭的直屬上司
- 主角要帶走塔利，需要先了解哈維的性格：「他不喜歡麻煩，但也不喜歡損失。」
- 主角的解決方式：找到一個替代塔利崗位的人，讓哈維覺得不虧本，順利讓塔利離開

**互動節點**：
- 第1卷第8章附近：主角第一次進格倫城碼頭，在找塔利前先觀察了哈維一個下午。

---

### 連恩（Lenn）——格倫城小型商行老闆

| 項目 | 內容 |
|------|------|
| 年齡 | 52歲 |
| 職業 | 雜貨商行老闆，兼接收一些「不方便問來源」的貨物 |
| 性格 | 貪財但謹慎，不輕易信任新面孔 |

**對主角的意義**：
- 主角在格倫城的第一個「資金轉換點」
- 主角通過一次精準的天候預判（前世記憶），建議連恩提前備貨，讓他賺到一筆
- 連恩後來成為主角在格倫城的商業掩護之一（幫主角的交易走帳）

---

### 路卡（Luca）——格倫城集市的流動說書人

| 項目 | 內容 |
|------|------|
| 年齡 | 30歲 |
| 職業 | 說書人（在集市說故事，收聽眾扔的銅板） |
| 性格 | 見聞廣博，樂於分享，但會誇大細節 |

**對主角的意義**：
- 格倫城的「消息流通中心」——任何城裡發生的大事，路卡說書時都會提到
- 主角不直接從路卡那裡拿情報，但他說書的版本幫助主角理解「市井對局勢的讀法」
- 偶爾路卡說的某句話，會讓主角修正自己的情報判斷

## 功能性一次性NPC

### 前世貴族——引發父親致命意外的那位

**第一卷核心前置威脅**：
- 名字：德文・安（Devlan An），三等貴族，格倫城某庄園的繼承人
- 性格：跋扈，常以金錢解決麻煩，有打人習慣
- 為什麼會讓父親死：雇用車夫進行一次秘密運輸，路上遭人伏擊，車軸被人動過手腳——父親只是工具性的死亡
- 主角的干預：讓父親拒絕這次包車任務（以「車軸有問題要修」為由，讓父親推掉）
- 德文・安後來的下場：因捲入私鬥被降爵，流出格倫城，主角沒有再遇到他
