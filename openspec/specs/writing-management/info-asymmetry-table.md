# 資訊不對稱表

## 用途

記錄各核心角色對主角的「知道程度」。確保在寫作時，角色的反應和對話符合他們實際知道的信息。

## 使用規則

1. **每章寫作前確認**：寫到某角色時，確認他在那個時間點「應該知道什麼」
2. **對話一致性**：角色的問題和反應必須符合他們實際知道的信息
3. **更新**：每當有重要信息傳遞給某個角色，立刻更新對應的格子

## 永遠不知道的人

| 角色 | 永遠不知道的內容 |
|------|--------------|
| 班底（除塔利外） | 主角是重生者（塔利於第 1 卷 ch052 已知核心；其他人到第五卷或更晚才會接觸這個層次的真相） |
| 全部人（包括塔利） | 主角前世詳細經歷的全部細節——所有戰友的身份、前世的具體日記式記憶、每一個改變過的時間線節點。塔利知道核心（重生、石河谷死亡），但不知道主角具體記得多少 |
| 安塔公爵 | 他永遠無法理解主角的計劃規模有多大，也無法理解「一個普通出身的孩子」怎麼能做到這些 |
| 梅拉 | 她從不問，所以她永遠只知道她選擇知道的：「你去做你要做的事，你要回來。」 |
