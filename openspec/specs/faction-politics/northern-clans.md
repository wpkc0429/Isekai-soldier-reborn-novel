# 北方氏族聯盟設定

## 基本特性

北方氏族聯盟不是一個統一的國家，而是十二個大氏族組成的鬆散同盟，共享一條「任何一個氏族被外部滅族，其他氏族都必須響應」的盟約。

| 項目 | 內容 |
|------|------|
| 人口 | 約80萬（分散在北境高原和森林地帶） |
| 軍事 | 武者比例最高——幾乎每個成年男性都是輕度武者 |
| 弱點 | 無法長期維持大規模軍事行動，缺乏後勤體系 |
| 對外態度 | 孤立主義，不主動擴張，但對入侵者極度強硬 |

---

## 氏族對外關係

**對格雷亞帝國**：
- 帝國曾三次試圖征服北方氏族，均以失敗告終
- 現在的關係：帝國用商品換取「不騷擾帝國北境」的默契，但沒有正式協議
- 帝國的秘密操作：定期向部分氏族首領提供「特別商品」（金屬武器、奢侈品），試圖分化聯盟內部

**對岱隆王國**：
- 偶爾的小摩擦（邊境劫掠），但沒有大規模戰爭
- 交易：氏族用皮草、礦石換取岱隆的糧食和農具
- 葛魯公爵負責維護這條交易線

**對自由城邦**：
- 城邦是氏族最主要的外貿夥伴（中立，不試圖控制他們）
- 港灣城每年有兩次「氏族市集」，是氏族人最願意出現的外部場合

## 在石河谷之戰中的戰略角色

**帝國的期望**：
- 帝國希望在石河谷之戰期間，北方氏族「不要插手」
- 帝國的賄賂：向三個較大的氏族首領提供了豐厚的商品，換取「保持中立」的承諾

**主角的目標**：
- 不是讓氏族出兵幫岱隆（他們不會）
- 而是讓氏族不接受帝國的賄賂，保持真正的中立
- 只要氏族保持中立，帝國就必須在北境保留軍事力量，無法全力投入石河谷

**主角的操作**：
- 第三卷：主角通過碧海城的情報渠道，把「帝國正在賄賂你們氏族的事實」傳到其他氏族首領耳中
- 這在氏族聯盟內部引發了一場關於「誰接受了帝國的錢」的內部爭議
- 結果：三個接受賄賂的氏族被迫退還，帝國的北境策略失效

## 灰狼卡索（Kasso the Grey Wolf）

| 項目 | 內容 |
|------|------|
| 身份 | 北方氏族最強的獨立武者，不屬於任何氏族 |
| 年齡 | 主角第三卷接觸時約55歲 |
| 能力 | 覺醒者（體能系強化），是岱隆+氏族+城邦範圍內公認最強的個人武力 |
| 現狀 | 隱居在北境邊緣小鎮，拒絕所有勢力的邀請 |

**卡索的過去**：
- 曾是氏族的戰士，後因一場他認為「沒有意義」的戰爭而離開氏族
- 他的判斷標準：「值得為之而戰的事」——他拒絕了帝國、王室、貴族的所有邀請
- 他相信「強者不應該為強者服務，而應該決定自己保護什麼」

**主角說服卡索的策略**：
- 主角不告訴卡索「你應該保護岱隆」
- 主角告訴卡索：「帝國打贏之後，北方氏族是下一個目標，因為他們是帝國唯一沒有滲透的地方。」
- 加上：「我不要求你帶兵。我只需要你的名字出現在那裡——讓帝國不確定你在不在場。」

**卡索最終答應的原因**：
- 主角展示了自己的真實武力，讓卡索評估「這個人是認真的」
- 主角沒有撒謊，沒有高估局勢，也沒有誇大成功機率
- 卡索的回答：「你不是在告訴我我必須贏。你是在說如果我不去，連嘗試的機會都沒有。」

## 北方氏族的戰略意義小結

- **威脅程度**：對岱隆和帝國都是潛在威脅，但可管理
- **戰略用途**：讓帝國無法把全部力量集中在東境
- **主角的底線**：不試圖統治氏族，只讓他們「不站帝國那邊」
- **長期關係**：石河谷之戰後，卡索回到北境，主角和氏族建立了一個非正式的互不侵犯默契
