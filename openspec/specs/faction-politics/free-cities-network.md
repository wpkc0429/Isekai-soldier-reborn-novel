# 自由城邦聯盟——情報與商業網絡設定

## 城邦聯盟概述

五個城邦各有主導產業，形成一個鬆散但高效的商業共同體。任何一個城邦都不能單獨存活，所以「聯盟」是現實需要，不是意識形態。

| 城邦 | 主導產業 | 主角使用方式 |
|------|---------|------------|
| 鑄錢城（Mintholm） | 金融體系、貨幣兌換 | 資金放大、匿名財務操作 |
| 碧海城（Bluehaven） | 情報中介、消息買賣 | 情報套利、建立情報信用 |
| 劍域城（Blademere） | 武者培訓、傭兵輸出 | 招募可靠武者，建立直屬武力 |
| 港灣城（Harrow） | 遠海貿易、走私通道 | 貨物轉運、建立岱隆以外的資產 |
| 文書城（Archon） | 法律文件、合約公證 | 建立合法身份，確保交易有法律保障 |

---

## 城邦聯盟的政治立場

**核心原則**：「任何人做生意都歡迎，但誰要統治城邦聯盟，我們就跟誰為敵。」

- 對格雷亞帝國：做生意，但拒絕政治附庸
- 對岱隆王國：交易夥伴，保持距離
- 對北方氏族：提供物資換取皮草和礦石，互利

**城邦聯盟的隱憂**：
- 帝國的商業滲透已深入鑄錢城（部分大商行有帝國資本）
- 碧海城的情報網絡被至少三個勢力同時使用，信息安全成問題
- 劍域城的武者輸出如果帝國大量採購，聯盟的獨立軍事力量將受損

---

## 主角在城邦聯盟的活動路線

### 第二卷：建立身份
- 以「商人」身份進入鑄錢城，建立第一個假名帳戶
- 在碧海城提供幾次精準情報（前世記憶），建立「這個人的情報很準」的口碑
- 目標：在城邦聯盟中有一個「可信的商業身份」

### 第三卷：建立工具
- 在鑄錢城建立資金放大管道：以情報套利積累的資金，投入城邦的金融體系
- 在碧海城建立雙向情報流通：主角賣情報，也買情報
- 在劍域城識別並接觸潛在的核心武者（後期成為直屬部隊的核心）

### 第四卷：石河谷之戰的城邦角色
- 主角的後備資金藏在鑄錢城的匿名帳戶，無論岱隆局勢如何都能存取
- 港灣城提供了一個「如果事情最壞」的逃跑通道（主角不打算用，但必須有）
- 碧海城的情報讓主角提前三個月知道帝國的石河谷計劃

## 城邦聯盟的關鍵NPC

### 布鮮（Buxen）——碧海城情報中介

| 項目 | 內容 |
|------|------|
| 年齡 | 主角進城邦時約50歲 |
| 職業 | 碧海城最大的情報中介商行老闆 |
| 性格 | 不關心情報的內容，只關心情報的「價值」 |

**對主角的功能**：
- 主角的前幾次情報「投資」讓布鮮對他刮目相看
- 布鮮不追查情報來源（那是他的職業倫理），但他給主角開了一個「固定客戶」的待遇
- 布鮮的中介網絡是主角了解帝國動向的最重要非直接渠道

### 卡爾・明特（Karl Mint）——鑄錢城金融商行

| 項目 | 內容 |
|------|------|
| 年齡 | 主角進城邦時約42歲 |
| 職業 | 鑄錢城第三大商行的二代繼承人 |
| 性格 | 精於數字，但有「只要合法就行」的道德底線 |

**對主角的功能**：
- 主角的資金通過卡爾的商行走帳，表面上是商業投資
- 卡爾不問主角的真實目的，只要回報穩定且合法
- 第三卷後期，卡爾開始懷疑主角的「投資」不只是商業行為，但他沒有主動調查

---

## 城邦聯盟對主角計劃的整體意義

城邦聯盟是主角在岱隆體系之外的「平行基礎設施」：

- **資金**：就算岱隆動亂，主角的財務基礎不會全部歸零
- **情報**：帝國的動向可以通過城邦渠道知道，不完全依賴岱隆內部情報
- **武力後備**：如果直屬武者不夠，劍域城有可以臨時動員的資源
- **退路**：港灣城提供了一個主角從不打算用、但必須有的出口

城邦聯盟不是主角的「大本營」，它是主角的「保險」。
