## 灰狼卡索（Kasso the Grey Wolf）

### 力量設定

| 項目 | 說明 |
|------|------|
| 等級 | 少數強者（等級5） |
| 覺醒類型 | 體能系（極致強化），感知系輔助 |
| 具體能力 | 速度達到常人50倍；力量足以徒手破開城牆；感知範圍100公尺 |
| 覺醒年齡 | 32歲（在一場他唯一一次「真正想贏」的戰鬥中覺醒） |

**限制**：
- 他的能力需要大量飲食維持（他一天的食物消耗是普通人的5倍）
- 長時間使用全力後需要長時間恢復
- 他的戰鬥方式是「終結者」——他進場，戰局結束

**在石河谷之戰中的用途**：
- 主角不讓卡索衝進正面戰場
- 卡索的任務：「讓帝國的指揮官知道你在場」——他的名字和他的出現本身就能讓帝國重新計算
- 卡索最終讓雷蒙轉向面對他，這讓帝國的正面推進節奏被打亂

## 黑鐵將 雷蒙（Reymon the Iron）

### 力量設定

| 項目 | 說明 |
|------|------|
| 等級 | 強覺醒者（等級4接近5） |
| 覺醒類型 | 體能系強化（主）＋感知系初期（副） |
| 具體能力 | 速度達到常人30倍；皮膚硬化（普通武器傷害大幅降低）；感知範圍50公尺 |
| 戰場功能 | 帝國的「展示品」——他的存在讓敵軍動搖 |

**心理特點**：
- 雷蒙享受被看見——他在戰場上是表演者，喜歡敵人的恐懼
- 這是他的弱點：他不喜歡「看不見他的戰鬥」
- 主角的策略：讓卡索佔據雷蒙的注意力，主角在背後做關鍵行動

**與主角的直接關係**：
- 主角在石河谷之戰中刻意不和雷蒙正面遭遇
- 他們的最近接近：雷蒙在20公尺外，主角用感知系確認了方向，繞開了

## 費格（Faeg）——安塔的護衛隊長

### 力量設定

| 項目 | 說明 |
|------|------|
| 等級 | 菁英武者（等級3頂端） |
| 特點 | 極度精煉的武技，是「人類體能上限的武者」 |
| 戰鬥風格 | 乾淨、有效、沒有多餘動作——職業殺手的風格 |

**與主角的對決（第4卷）**：
- 費格接到命令「解決主角」，帶五個武者級護衛出動
- 主角的感知系讓他在費格到達前就感知到了他們的接近
- 戰鬥結果：主角打敗費格，但付出代價（左臂傷）
- 主角沒有殺費格——「你活著比死了有用」
- 費格的心理震撼：他一生從未被人「讓活著」——那一刻讓他開始重新評估主角

**費格後來的發展**：
- 安塔公爵在第5卷崩潰後，費格選擇離開安塔家族
- 費格沒有加入主角，但也沒有成為敵人——他以一個傭兵的身份在城邦活動
- 主角：「費格這種人，讓他自由比讓他欠我更有用。」

## 陳卡（Chen Ka）

### 力量設定

| 項目 | 說明 |
|------|------|
| 等級 | 菁英武者（等級3中段） |
| 特點 | 軍人式的平衡型武者——個人戰鬥力強，但更擅長帶兵指揮 |
| 本世年齡 | 石河谷之戰時約52歲，體能開始走下坡但經驗彌補 |

**在石河谷之戰中的角色**：
- 陳卡帶領部分轉向主角的邊境軍（軍閥派中的關鍵部分）
- 他不是主角的武力，他是主角合法性的來源：「邊境軍指揮官認可這個人」
- 石河谷之戰的正面軍事佈局由陳卡和卡索負責，主角負責關鍵節點的直接行動

## 力量對比的戰略意義

主角設計他的行動時，始終把力量等級算入考量：

**主角的基本原則**：
- 能用情報和謀略解決的事，不動武
- 必須動武時，選擇「打得贏的對手」
- 打不贏的對手（雷蒙、卡索），讓他們「對著別人」

**石河谷之戰的力量分配**：
```
卡索 ← 牽制 → 雷蒙
陳卡 ← 正面對抗 → 帝國主力
主角 → 切斷通信 / 對決費格（側翼行動）
```

這個分配讓每個人都做「剛好能做的事」，而不是讓一個人扛所有。
