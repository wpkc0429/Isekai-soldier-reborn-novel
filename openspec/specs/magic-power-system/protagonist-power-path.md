# 主角力量成長路線

## 前世的力量基線

前世的艾登是個「有天賦但沒發展」的案例：

| 前世情況 | 說明 |
|---------|------|
| 武者覺醒年齡 | 16歲（比平均晚1年，因為沒有系統訓練） |
| 前世最高等級 | 武者（從未達到菁英武者） |
| 戰鬥風格 | 散兵式粗糙技法，靠先天感知彌補技術不足 |
| 前世後期收穫 | 21歲遇到老武者，得到「武術框架的概念」，但沒有時間真正實踐 |
| 死亡時的狀態 | 普通武者，不算出色的士兵 |

**前世的教訓**：
> 他在死前才明白——感知天賦是他一直有的東西，但他從來不知道怎麼用它。如果當時有系統……

## 本世的成長路線

### 先天優勢（重生帶來的）
1. **空間感知天賦**：前世就有，本世更早知道如何利用
2. **老武者的武術框架**：前世後期學到的概念，現在從8歲開始系統應用
3. **前世8年的戰場肌肉記憶**：身體是孩子的，但動作模式的腦內印記仍在

### 本世的覺醒時間線

| 年齡 | 里程碑 | 說明 |
|------|--------|------|
| 8-10歲 | 熟悉兒童身體，隨手練習 | 前世肌肉記憶讓動作比同齡孩子協調，但體力是孩子的上限 |
| 10-12歲 | 開始按老武者框架系統練習 | 不找師傅，用前世記憶中的框架自練，打好架構 |
| 12歲 | 武者覺醒 | 比前世提早4年；體能達到常人2倍範圍 |
| 12-16歲 | 武者整合期，刻意隱藏 | 和塔利當訓練搭檔，偶爾刻意輸給塔利保持偽裝 |
| 16歲 | 菁英武者 | 老武者框架完全整合進身體；一敵十的真實水準 |
| 16-18歲 | 感知初期徵兆 | 覺醒前的前兆：感知微妙增強，不確定是什麼 |
| 18-19歲 | 覺醒者覺醒（感知系） | 刺殺觸發；初期15公尺範圍感知 |
| 19-21歲 | 覺醒整合 | 感知範圍擴展至30公尺；老武者框架讓整合速度加快 |
| 21歲後 | 戰場感知成熟 | 能提前0.5-1秒感知攻擊方向；石河谷之戰的個人行動基礎 |

## 各卷力量定位

| 卷 | 等級 | 公開程度 | 實際用途 |
|----|------|---------|---------|
| 第1卷 | 准武者→武者 | 「身手不錯的孩子」 | 自保，保護父親，沒有公開戰鬥 |
| 第2卷 | 武者 | 隱藏武者等級 | 訓練班底，偶爾自保，不在政治場合亮武力 |
| 第3卷 | 菁英武者 | 卡索知道（無法隱瞞），其他人不確定 | 說服卡索的談判籌碼之一 |
| 第4卷 | 覺醒者 | 戰場行動後成為「公開的秘密」 | 切斷帝國通信樞紐；費格對決 |
| 第5卷 | 覺醒者 | 已知，不再隱藏 | 退居幕後，開始整理武學框架用於傳承 |

## 主角武力的刻意限制

主角知道「展示武力」會讓所有人重新評估他，所以他有嚴格的展示規則：

**不展示的情況**：
- 任何可以用其他方式解決的場合
- 政治談判（武力應該是談判籌碼，不是展示品）
- 班底訓練（他刻意讓塔利認為差距不那麼大）

**展示的情況**：
- 面對卡索——因為卡索能評估真實武力，主角選擇直接坦誠
- 費格對決——因為那是不得不的直接衝突
- 刺殺後的場合——有時被動曝光

## 力量成長的敘事意義

前世的艾登「有天賦，但死前才開始真正發展」。

本世的故事不是「主角變強→用強大的武力解決問題」，而是：

> **一個從前世就有天賦、但從未有機會發展的人，重生後選擇了正確的方式，在剛剛夠用的時間內讓自己足夠強——不多，但剛好夠做他需要做的事。**

他不是全書最強的人（那是卡索或帝國的幾個強者），但他是「用力量最有效率的人」——他知道什麼時候用武力，什麼時候絕對不用。
