# 第五十二章：你想幹嘛

| 項目 | 內容 |

| 章節標題 | 你想幹嘛 |
| 帝曆時間 | 帝曆262年，秋 |
| 主角年齡 | 12歲 |
| 節奏標記 | ▲ （塔利第一次直接問動機，關鍵對話） |
| 目標字數 | 2,500字 |
| 實際字數 | 約2,500字 |

---

塔利說：「你到底想幹嘛。」

這個問題沒有前言，沒有引子。他們剛做完一組試手，在院子裡坐著，艾登在整理訓練記錄，塔利喝水，然後說了這句話。

艾登把記錄停下來，看了他一眼，說：「你這個問題問的是哪一層？」

「什麼哪一層，就是這個問題，你想幹嘛，你這四年做的那些事，我跟著你，我知道有個方向，但你從來沒說清楚。」

「你說說你知道什麼方向。」

塔利想了一下，說：「你說有一場仗要打，在十幾年後。你說讓該活著的人活著。你讓我來這裡，讓我練，讓我在塔恩鎮站穩腳跟。你在格倫城有幾個生意上的聯繫，你在觀察那個藥鋪的學徒，你讓我知道了安塔公爵的名字。」他停了一下，說，「這些我都記著，但我不知道這些東西指向什麼。」

---

艾登把記錄放下，想了一下，說：「你說你想知道什麼——你想知道我的計劃，還是你想知道我為什麼這樣做？」

塔利說：「兩個都說。」

「計劃我不能全說，不是不信任你，是有些部分我自己還沒確認，說了之後如果有一步出了問題，讓你帶著一個錯的信息比帶著不完整的信息更危險。」艾登說，「但為什麼做，這個我可以說。」

塔利說：「說。」

「讓該活著的人活著，」艾登說，「這句話我說過，這是我全部行動的第一個前提。」

「然後？」

「然後這場仗是真的，十一年後，不是我猜的，是我知道的，確實知道。這場仗如果沒有人提前準備，死的人會很多，包括你，包括一些本來不應該死的人。」

塔利說：「你怎麼知道？」

「前世的記憶，」艾登說，不猶豫，「我是重生的。」

---

院子裡安靜了幾秒。

塔利沒有說什麼，他把那句話在他腦子裡過了一遍，然後說：「重生是什麼意思，你死過一次，然後回來了？」

「對，在石河谷的溪水裡，二十二歲。然後我在這個身體裡八歲的時候醒來。」

「你前世的那個時候，就是十一年後那場仗結束後？」

「在那場仗裡，不是結束後。」

「你死了，就是輸了？」

「那場仗輸了，很多人死了。」艾登說，「包括你。你那一世是我的副手，你死在那個冬天的石河谷，我把你推進水裡讓你往下游游，但你還是死了。」

塔利說：「你推我進水裡了？」

「讓你往安全的方向游，你沒有游到。」

塔利想了一下，說：「那不是你的問題。」

「我知道，但你死了。」

「那是前世，我這世沒死，」塔利說，「你要我做什麼？」

---

他看著塔利說這句話。

「你要我做什麼」——不是「我不信你」，不是「你說的太離奇了」，是直接跳到「那我要做什麼」。這個反應他沒有預期，但又很像塔利——他判斷了「這個人說的是真的，所以我問下一個問題」，從聽到問題到問下一個問題之間，他就給了一個判斷，沒有浪費時間在懷疑上。

艾登說：「你現在做的那些就是了，練，在塔恩鎮站穩，跟著我，等時候到了，我說，你做，就這樣。」

「就這樣？」

「就這樣，現在就這樣，後來有後來的說法，我現在只說現在的。」

塔利說：「你說你前世二十二歲死的，你現在十二歲。」

「對。」

「那你這一世，就是在做你前世來不及做的事？」

艾登說：「有一部分是，有一部分不是，不是每件事都是補做前世沒有做的，有些事是這一世應該做的。」

塔利點了點頭，說：「那你說你知道那場仗的結果，是因為你在那場仗裡死過？」

「對，我知道那場仗是怎麼打的，知道結果，知道誰死了。這一世，我要讓它走不同的方向。」

塔利說：「你覺得你能做到？」

---

艾登說：「不知道，但我要試。前世我是一個沒有什麼資源的士兵，對著一個已經成形的局面，我改不了什麼。這一世我有十一年的準備時間，我從八歲開始，現在十二歲，我在積累，我在讓條件往對的方向走。」他說，「我能不能做到，取決於我做了多少和那場仗的實際走向之間的差距，這個差距我現在說不準。」

「但你要試。」

「對。」

塔利說：「那我跟著你。」

他說這句話的語氣沒有任何勉強，是一個他想清楚了然後說出口的確認——不是「好吧那我就繼續跟著」，是「我確認了，我跟著你」。

艾登說：「我知道，謝謝。」

塔利說：「不用謝，前世你推我進水裡讓我往下游走，那我就繼續游就是了。」

然後他站起來，說：「我去倒水，你繼續記你的東西。」

---

院子裡，秋天的光往低處走了，樹影斜在地上。

他把今天說的話想了一遍，確認了哪些說了，哪些沒說——他說了重生，說了前世，說了石河谷，說了他要做的事的方向，但他沒說他覺醒了，沒說艾拉和賽門，沒說安塔公爵的具體計劃，沒說他知道的更多細節。

這是他第一次對任何人說出「重生」這個詞。

他想了一下，這個時機對嗎。

他的判斷是對的。塔利問了「你想幹嘛」，這個問題是一個需要給一個核心答案的問題，如果他繼續含糊其詞，塔利的信任就會有一個缺口，那個缺口會隨著時間讓他的執行變得困難——一個不知道為什麼的人，和一個知道了核心的人，他們能做的事不一樣。

現在的說法是塔利能接受的最低信息量，夠他確認跟著艾登是對的，不超過他現在需要知道的。

剩下的，等時候到了。

塔利回來，把一杯水放到他旁邊，說：「繼續記你的東西。」

「好。」

他繼續記，秋天的院子安靜，那個說出口的「重生」在他心裡待了一下，然後沉下去了。
