# 第十四章：進城

| 項目 | 內容 |
|------|------|
| 卷次章節 | 第一卷第十四章 |
| 章節標題 | 進城 |
| 帝曆時間 | 帝曆259年，冬初 |
| 主角年齡 | 9歲 |
| 節奏標記 | ▲ （上升，第一次接觸更大世界） |
| 目標字數 | 3,000字 |
| 實際字數 | 約3,000字 |

---

父親說要進格倫城，說要帶他一起去。

這不是一個他需要努力才得到的機會——父親說有一個固定的格倫城客人要他去取一批家具，車要用大半天，他說「你沒有去過城裡，要不要一起去看看」，說話的方式是隨口提的，不是特別鄭重的事情。

艾登說：「好。」

他沒有表現出他已經等這個機會等了很久了。

---

進格倫城的路走了大半天。

父親趕車，艾登坐在車廂裡，偶爾掀開布簾往外看。路從土路變成碎石路，再變成官道，車輪的聲音隨著路面的變化改變。沿途的村落越來越密，遠遠地可以看到城牆的輪廓，然後越來越清晰。

格倫城的城牆是青石砌的，高度比他在前世見過的邊境哨站要高出很多，走進去要過一個城門，城門的衛兵在收進城費，每輛車三個銅板，每個人一個銅板。父親很熟練地把銅板遞出去，說「例行的，進城都這樣」。

艾登沒有說話，把城門的規模、守衛的人數、收費的方式記下來。

他不是要用這些，他只是習慣性地記所有的東西。

---

城裡比他預期的要嘈雜。

塔恩鎮有兩百多戶，格倫城是郡城，幾千戶，街道上的人流密度完全不同。他在塔恩鎮走路可以很清楚地感覺到自己在哪裡，但在格倫城他需要花更多的注意力在定位上。父親熟悉這裡，走的路線很穩定，他跟著父親走，同時在腦子裡建立一個粗糙的城市地圖。

他們先去客人的地方交接，那是在城東的一條商業街，客人是一個做布料生意的商人，父親幫他搬了箱家具進去，收了款，在店門口說了幾句話。那個商人瞄了艾登一眼，說「你兒子？」，父親說「對」，那個商人說「眼神跟你不像，倒像他媽」，父親說「他媽說話精明，他隨她」，那個商人笑了一下。

艾登站在旁邊，讓自己的臉保持一個聽不懂大人說話的孩子的表情。

---

接完任務之後父親說有點時間，帶他在城裡走了一圈。

他們走過集市區，那裡比塔恩鎮的集市廣場大了幾倍，攤位密集，品種從食物到工具到奢侈品都有。父親說「格倫城的集市每天都開，不像塔恩鎮三天一次」，艾登把這個記下來。

他們走過碼頭方向，但沒有走近——父親說那邊人多，帶著孩子不方便。艾登看了一眼遠遠的碼頭方向，看到了一些在搬貨的人的背影，他沒有分辨出任何人的臉，距離太遠了。

*塔利在那裡。*

*十三歲，還有一年多輪軸就會壞。*

他讓這個念頭在腦子裡停了不到三秒，然後把它壓下去。

他今天的目的不是找塔利，他今天只是第一次進城，看，記，建立基礎。找塔利要等他在城裡有足夠的活動能力，不是今天。

---

他們走過了一條看起來比較安靜的巷子，父親說那邊是一些中等規模的貿易商行。艾登看了一眼巷子裡的招牌，在一個看起來不算新的招牌上看到了一個名字：連記雜貨行。

連記。

他把這個名字和他從母親那裡聽到的、夏天那個有問題的格倫城商人連姓商人對了一下。這個位置說明那個商行還在開，沒有因為合夥人糾紛就倒閉。

他在這條巷子多看了兩秒，然後繼續跟著父親走。

---

父親在城裡的某個地方停下來喝了杯茶，讓艾登坐在旁邊，說：「你今天看了什麼。」

這個問法不是隨口的，是一個真的在問的問題。

艾登說：「比塔恩鎮大很多。人也多，道路好一點，城門收進城費，碼頭的聲音從那邊就能聽到。」

父親說：「還有呢。」

「布料商在城東，你說的那條商業街，看起來專門做給外地客人的生意，本地人大概用比較固定的地方。」

父親停了一下，說：「你怎麼看出那是做外地客人的生意。」

「那條街的招牌比較大，字寫得清楚，陳列方式讓人一眼就能看到貨品，不像鄰鎮熟識的店不需要這樣。而且在城門進來的方向是直線，外地人進城最容易走到那裡。」

父親喝了口茶，說：「你說得不錯。」

然後他沒有說更多，但那個「不錯」讓艾登知道這個觀察是對的。

---

回程的車上，他讓自己把今天看到的東西重新過了一遍。

城門，收費，守衛。主街，布料商的方向，碼頭的方向。連記雜貨行，在比較安靜的巷子裡，還在開。

他看到了格倫城，但他還沒有看進格倫城。這是第一次，他知道下一次可以看更深一點。

他最大的問題不是看到了什麼，是他現在的限制：他沒有辦法在格倫城獨立行動，他依賴父親帶他進去，而父親的行程是由他的客人決定的，不是由艾登決定的。

他需要找到一個合理的理由讓自己能更頻繁地進城。

這個問題他現在還沒有答案，但他知道那個答案存在，等他想到了就用。

父親的馬車在官道上穩穩地走，車輪的聲音從格倫城的石子路換回了泥土路的低沉聲響，塔恩鎮在遠處漸漸近了。

艾登閉上眼睛，讓那個嘈雜的、擁擠的、比他估計更複雜的城市在腦子裡安靜地待著。

他以後還會回去。

而且會更多次。
