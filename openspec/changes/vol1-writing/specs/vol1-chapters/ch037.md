# 第三十七章：你是猜的還是算的

| 項目 | 內容 |
|------|------|
| 卷次章節 | 第一卷第三十七章 |
| 章節標題 | 你是猜的還是算的 |
| 帝曆時間 | 帝曆262年，春 |
| 主角年齡 | 12歲 |
| 節奏標記 | ▲ （父親第一次直接質問，主角壓力上升） |
| 目標字數 | 2,500字 |
| 實際字數 | 約2,500字 |

---

父親問他這個問題，是在春天一個普通的傍晚，兩個人坐在院子裡，艾登在整理一批要給班尼的草藥記錄，父親在磨刀。

「艾登，」父親說，「我問你一件事。」

「說。」

父親把磨刀石放下，看了他一眼，說：「你是猜的，還是算的？」

---

他沒有立刻回答，把手上的記錄繼續寫完一行，然後才說：「什麼意思？」

「我說的是，你說的那些，老傅的事，那條路，還有後來的那幾次——」父親停了一下，然後說，「連格倫城那邊的客源，我後來問了，是有人介紹的，但那個人說是你讓他介紹的，你讓他說是別的路子介紹的。」

艾登說：「哦。」

他知道這一天會來，他不確定是這個時間點，但他知道父親遲早會拼起來的——父親不笨，他只是習慣不問，但積累到一定程度，他就會問。

父親說：「你說是猜的，但你的猜準的出奇。我說的不是一次兩次，是從你八歲到現在，每次你說的話後來都有個結果在那裡等著。老傅的事，馬爾坡的路，連記那邊的商隊消息，還有那個塔利——」他說到這裡停了一下，「那個孩子你說他做這個合適，他真的做得挺好的。」

院子裡安靜了一段。

---

艾登把記錄放下，說：「就是猜的。」

父親看著他，說：「就是猜的。」

「對。」

「一次猜中是運氣，」父親說，「十次呢？」

「十次說明我猜的方向不差。」

父親沉默了一會兒，說：「你說話的方式從小就不一樣，我和你媽說過，你媽說是讀書讀多了，老氣了。我一開始也這樣想。後來覺得不對，不是老氣，是另一種東西。」

艾登說：「什麼東西？」

「就是……」父親停了一下，「你說話，你知道你說的話的重量。一般人說話，不管是大人還是小孩，說了不一定知道這句話是什麼意思，但你說的每一句，你自己先清楚了，才說出來。這個不是讀書讀來的，是另一種事情。」

---

艾登看著父親。

父親是個車夫，不是一個擅長說話的人，但他剛才說的那句話很準確——「你說的每一句，你自己先清楚了，才說出來」——這是一個觀察了四年才說出口的東西，不是隨口說的。

他想了一下，說：「你說的對。」

父親說：「那你說，你到底怎麼知道的。」

「我看的，聽的，想的，」艾登說，「老傅那件事，我那時候問過老德，馬爾坡的路他說過不好走，我又聽過你說那個貴族換了車夫，我把這兩件事放在一起，覺得不對，就說了。」

「你八歲說的那些。」

「八歲也會聽東西，」他說，「我聽得比較仔細。」

父親說：「嗯。」

他沒有繼續問，把磨刀石重新拿起來，繼續磨，但手的力道換了一個節奏，不是放下了，是在想。

---

艾登讓父親想了一段，然後說：「你現在接的客人，裡面有幾個是格倫城做布匹生意的？」

父親說：「兩個，都是連著跑的。」

「他們說的路線你有沒有注意是哪一條？」

「北路，為什麼？」

「北路今年下半年有一段要修，他們的商隊如果提前不知道，到時候繞路要多出三天。你現在知道，可以提前給他們說，他們會記你的情。」

父親看了他一眼，說：「你又猜的？」

「老德說的，他說鎮長那邊有個計劃，但還沒公布，估計秋天會公布，到時候要修了再改路已經來不及了。」

父親把磨刀石放下，想了一下，說：「那我下次去格倫城，提前說一聲。」

「對。」

父親說：「……你的猜，準的出奇。」

他說這句話的語氣和剛才不一樣了，沉了一些，像是在確認一件他一直不確定的事。

---

那天晚上，父親沒有再問，但他和母親說了什麼，艾登在屋裡聽到了幾個字：「那個孩子……不是普通的猜……」

後面的話聲音輕了，他沒有聽完。

他躺在床上，把天花板看了一會兒。

父親接受了「猜的」這個說法，但他不完全相信。他現在對兒子的判斷的態度變了——以前是「讓他說說看」，現在是「他說了再考慮考慮」，這是一個重要的變化。

這個變化，艾登需要，但也需要控制。

父親如果把他的判斷當作「一定準的」，他就失去了一個緩衝——他的判斷不是一定準的，他只是準的比一般人高。如果父親把他說的話當作神諭，那父親在不應該跟著走的時候也會跟著走，這個風險比父親不信他的風險更高。

他要讓父親相信他，但不是百分百相信他，是「我說的值得認真考慮」這個程度。

這個度，還要繼續調。

窗外，春天的蟲子叫了起來，院子裡的燈還亮著。

父親比前世活著，這個已經不需要擔了。接下來要做的，是讓他活得更好。
