# 格雷亞帝國擴張策略

## 帝國的基本立場

格雷亞帝國不急。它的擴張策略是「讓目標自己腐爛」——通過滲透、分化、經濟控制，讓目標國家在沒有戰爭的情況下逐步失去自主性，然後在最合適的時機正式吞併。

**帝國的核心邏輯**：
- 軍事征服成本高，且會激起抵抗
- 「主動投誠」的成本最低，而且被吞併後更容易管理
- 所以帝國總是先找代理人，讓代理人替他做骯髒的事

---

## 帝國對岱隆的長期佈局

### 第一階段：滲透期（帝曆200-240，已完成）
- 通過商業渠道控制岱隆部分礦山和港口的貿易定價權
- 在岱隆貴族中扶植親帝國勢力（安塔公爵是最成功的案例）
- 讓岱隆王室財政對帝國商業貸款產生依賴

### 第二階段：分化期（帝曆240-260，進行中）
- 利用貴族派對王室的不滿，讓岱隆中央無法形成統一應對
- 在邊境製造「安全威脅」，迫使岱隆需要帝國的「保護」
- 石河谷關口的戰略地位被識別——那是岱隆東境防線的咽喉

### 第三階段：吞併（帝曆261預計）
- 石河谷之戰：帝國計劃利用安塔公爵的配合，消滅第七邊境軍
- 邊境防線崩潰後，帝國以「恢復秩序」為名進入東境
- 安塔得到自治協議，但那只是過渡安排（帝國從未打算長期遵守）

---

## 帝國的石河谷戰役視角

**帝國的目標**（不是簡單的領土）：
1. 消滅第七邊境軍——岱隆唯一有實戰經驗的邊境主力
2. 打垮岱隆的抵抗意志——讓其他邊境軍看到「抵抗無用」
3. 讓安塔公爵承擔「背叛盟友」的罵名，帝國以「中立調停者」出現

**帝國使用的代理人**：
- 安塔公爵：內部配合，製造第七邊境軍的孤立
- 一名帝國特使（拉克・維恩，Lak Vein）：在岱隆王都游說，讓王室在戰役前夕不增援

**帝國的備選方案**：
- 如果石河谷之戰失敗：暫退，等下一個機會
- 帝國的長線思維：「等不到十年，岱隆自己會崩潰」

## 帝國在石河谷之戰的軍事佈置

| 部分 | 內容 |
|------|------|
| 主力 | 第三帝國邊境軍團，約12,000人 |
| 特殊單位 | 黑鐵將 雷蒙（Reymon）率領的覺醒者護衛隊，約300人 |
| 策略 | 利用石河谷地形，前段引誘岱隆軍入谷，後段封口 |
| 預期結果 | 三天內殲滅第七邊境軍全部8,000人 |

**黑鐵將 雷蒙（Reymon）**：
- 帝國最知名的覺醒者將領，感知系強化＋體能系強化雙線覺醒
- 在石河谷之戰中是主角直接對抗的邊界——主角的目標不是打贏雷蒙，而是繞開他
- 雷蒙的特點：他知道自己是帝國的「戰場展示品」，他的存在讓敵軍動搖，他享受這個角色

## 帝國視角下的主角威脅等級

| 卷 | 帝國對主角的評估 |
|----|---------------|
| 第1-2卷 | 不知道主角存在 |
| 第3卷 | 帝國情報網察覺「有人在系統性干擾東境的滲透工作」，尚未確認身份 |
| 第4卷 | 石河谷之戰後，主角的存在正式進入帝國情報體系——評估為「區域性威脅，需要處理」 |
| 第5卷 | 帝國在東境的吞併計劃因主角的行動延誤5-10年，開始認真對待 |

**帝國的最終決定**：
- 石河谷之戰結束後，帝國沒有立即再動手——因為代價超出預期，加上北方氏族的不確定性
- 帝國選擇先鞏固已吞併的地區，再做下一步
- 這個「延遲」給了主角和岱隆足夠的喘息空間——這也是主角計劃中最重要的一環

## 主角對帝國的策略

**核心原則**：
> 主角的目標不是打敗帝國，而是讓帝國覺得吞併岱隆的成本超過收益。

**具體手段**：
1. 讓帝國在石河谷之戰的代價超過預期（改變全軍覆沒的結局）
2. 讓北方氏族保持中立（帝國一旦兩線作戰，時間表被打亂）
3. 建立一個「帝國不能輕易消滅的後方根基」
4. 讓安塔公爵的配合失效（沒有岱隆內部的配合，帝國的成本急升）

**主角不打算做的事**：
- 不試圖征服帝國
- 不試圖建立一個「足以正面對抗帝國的聯盟」
- 只需要讓帝國得出「現在動手不划算」的計算結論，然後讓岱隆有機會自我修復
