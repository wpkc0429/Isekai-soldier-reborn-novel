# 感情線時間線

## 概述

這份文件記錄全書三條感情線的演進：
1. **主線**：梅拉與主角的關係
2. **家人線**：主角與家人（父母、妹妹）的情感連結
3. **夥伴線**：班底成員信任關係的建立過程

---

## 一、梅拉×主角感情線

### 關係演進總覽

| 卷 | 主角年齡 | 梅拉年齡 | 關係階段 | 關鍵節點 |
|----|---------|---------|---------|---------|
| 第1卷 | 8-12歲 | 9-13歲 | 鄰居→玩伴 | 第一次對話；「你很怪但不是壞人」 |
| 第2卷 | 12-16歲 | 13-17歲 | 察覺→默契初始 | 梅拉開始知道主角有秘密但不問 |
| 第3卷 | 16-21歲 | 17-22歲 | 沉默的共識 | 「你去做你要做的事，但你要回來」 |
| 第4卷 | 21-24歲 | 22-25歲 | 等待→重逢 | 戰後「你受傷了，進來我幫你看」 |
| 第5卷 | 24-29歲 | 25-30歲 | 婚姻→家 | 求婚；生子；最後一幕 |

---

### 感情線關鍵場景

**第1卷：第一次真正的對話**
- 梅拉：「你說話很奇怪，你到底在想什麼？」
- 主角沒有回答
- 梅拉：「算了。你怪歸怪，但不是壞人。」
- 功能：建立梅拉「不繞彎、說真話」的性格基調

**第1卷末：主角的困惑**
- 主角意識到梅拉是他「唯一沒有評估框架的人」
- 這讓他困惑——他不知道這種困惑的意義

**第2卷：梅拉不追問**
- 梅拉看見主角在觀察一個商人，問了，主角敷衍
- 梅拉：「算了，反正你之後會讓我知道的。」然後走開
- 功能：梅拉的信任方式是「不問，但記著」

**第2卷：野花場景**
- 主角剛做完一個艱難決定，梅拉拿著野花出現
- 她沒有問任何事，只說了鎮上的瑣事
- 主角手裡握著花，一句話都說不出來
- 功能：第一次讓主角無法「分析場景的用意」

**第3卷：「你要回來」**
- 主角在塔恩鎮的時間越來越少
- 某次回來，梅拉說：「你要回來。」
- 功能：全書情感線最接近告白的一句話，但兩人都沒有說破

**第4卷：戰後重逢**
- 主角回來，身上有傷
- 梅拉沒有哭，沒有說「你終於回來了」
- 她說：「你受傷了，進來我幫你看。」轉身進門
- 功能：梅拉的情感表達永遠是「做點具體的事」——這就是她的愛的語言

**第5卷：求婚**
- 普通的傍晚，藥鋪門口
- 「我想讓你嫁給我。不是因為現在方便，也不是因為這對什麼計劃有用。就是……我想。」
- 梅拉：「你終於說了一句不計算的話。」然後點頭
- 功能：主角生命中唯一一次「沒有策略性目的」的重大決定

---

### 感情線的敘事原則

**為什麼不早告白**：
- 主角不是不知道自己的感情——他只是無法在「事情結束之前」讓任何事成為拖累
- 梅拉不是在等他——她只是選擇「先讓他去做他要做的事」

**梅拉的功能不是「可愛的戀人」**：
- 她是主角在整個計劃執行過程中唯一的「現實感錨點」
- 她的日常讓主角記得他做這些的初始目的

## 二、家人感情線

### 父親（羅爾）的情感演進

| 卷 | 節點 |
|----|------|
| 第1卷 | 父親第一次照兒子的意思做決定，後來理解了那個決定的重要性 |
| 第2卷 | 父親接到更好的訂單，開始相信兒子的判斷 |
| 第3卷 | 父親已不再問理由，選擇「跟著走，不多問」 |
| 第4卷 | 「你去的地方，我不問了。但你要回來。」（和梅拉說了類似的話——兩個愛他的人） |
| 第5卷 | 老了一些，知道兒子做了很多他不完全理解的事，選擇不問。只說：「你做到了。」 |

### 母親（艾拉）的情感演進

| 卷 | 節點 |
|----|------|
| 第1-2卷 | 注意到兒子越來越「不像小孩」，選擇不追問，但開始有意識地說更多閒話（她覺得他在學） |
| 第3卷 | 家境好轉，碎念中開始出現「謝天謝地」 |
| 第4卷 | 兒子去打仗前，她沒有哭，只問：「你有沒有帶夠吃的？」 |
| 第5卷 | 不再擔心柴米油鹽，偶爾還是碎念，但大家都知道那是她的愛的語言 |

### 妹妹（莉亞）的情感演進

| 卷 | 節點 |
|----|------|
| 第1卷 | 每天跑去找哥哥，不明白他為什麼偶爾「突然看她看很久」 |
| 第2-3卷 | 開始有自己的想法，問過幾次「你去外面做什麼」，哥哥沒有回答 |
| 第4卷 | 哥哥告訴她「保護好爸媽」，她答應了，然後看著他走 |
| 第5卷 | 出嫁，她的另一半是主角觀察了很久才認可的人。孩子在院子裡跑——那是最後一幕。 |

## 三、夥伴信任建立時間線

### 塔利的信任演進

| 節點 | 說明 |
|------|------|
| 第1卷第25章 | 第一次見面，主角的行動讓他覺得「這人有兩把刷子」 |
| 第1卷末 | 主角帶他離開碼頭，給他一個出路——這是信任的起點 |
| 第2卷 | 塔利武者覺醒，主角在場，那是兩人之間一個無聲的分水嶺 |
| 第3卷 | 刺殺發生，塔利在旁邊，兩人的配合第一次到達真正的默契水準 |
| 第4卷 | 石河谷之戰前：「你確定我們能活著回來？」信任轉化為「把命交給你的決定」 |
| 第5卷 | 深夜問話：第一次主動問出主角的核心，主角第一次說了真話的一部分 |

### 賽門的信任演進

| 節點 | 說明 |
|------|------|
| 第1卷 | 主角讓賽門「意外」得到更好的機會——賽門感覺到有人在幫他，但不確定是誰 |
| 第2卷 | 確認是主角後：「我需要知道的你會告訴我」——這是賽門的信任方式 |
| 第3卷 | 賽門發現主角資金漏洞，主動解決——這是他第一次「主動保護主角的利益」 |
| 第5卷 | 共同管理戰後的財務和情報網絡，信任已是「默認基礎」 |

### 艾拉・布蘭的信任演進

| 節點 | 說明 |
|------|------|
| 第1卷 | 主角認出她，保持距離，只說「那個秘密，有一個人不在意」 |
| 第2卷 | 觀察期：她在等主角展示「這個人值不值得信任」 |
| 第2卷末 | 「我決定相信你了。」——艾拉的信任是主動給出的，不是被說服的 |
| 第3卷 | 第一個知道主角覺醒的人，她選擇不說，讓主角自己決定告訴誰 |

### 凱斯的信任演進

| 節點 | 說明 |
|------|------|
| 第2卷 | 主角解決了凱斯的麻煩，然後沒有要求任何回報 |
| 第2卷 | 凱斯追上去的那一步，是信任的起點 |
| 第3卷 | 「我知道你沒告訴我全部，這樣反而對我在場合裡說話有保護」——他理解了主角的信息分層邏輯 |
| 第4卷 | 獨立完成任務後：「我現在才真的相信了。」——相信的是自己，也相信了主角的判斷 |

### 尤拉的信任演進

| 節點 | 說明 |
|------|------|
| 第1卷末/第2卷初 | 三次測試通過後：「我認你這個人。」 |
| 第2卷 | 「合作的人」是她的官方說法 |
| 第3卷 | 「下次這種事先告訴我，我可以安排退路。」——這已不是純利益語言 |
| 第5卷 | 她建立自己的事業，但那個事業的基礎和主角不可分割——她不說，但她知道 |
