# 人物名冊（Character Registry）

## 說明

三個等級：
- **A級**：核心人物，有完整的角色卡，出現在每卷
- **B級**：重要的支線人物，有基礎設定，在1-2卷有重要戲份
- **C級**：功能性NPC，只需要基本資料，在特定場合出現

---

## 升降級規則

| 情況 | 操作 |
|------|------|
| C級NPC在多章出現並開始有個性 | 升B級，補充基礎設定 |
| B級角色在後期成為重要戰略夥伴 | 升A級，建立完整角色卡 |
| A級角色在某卷後戲份大幅減少 | 降B級，移至「退場角色」 |
| B/C級角色死亡或永久退場 | 移至「退場人物」欄 |

## 新人物登記程序

當一個新NPC在章節中出現並被給予名字時：
1. 立刻在C級名冊登記
2. 如果在2個以上的章節重複出現，評估是否升B級
3. C級NPC無需完整角色卡，只需名冊格式的基本信息
