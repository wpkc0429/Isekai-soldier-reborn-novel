# 世界力量等級表

## 五個層次

| 等級 | 名稱 | 占人口比例 | 特徵描述 |
|------|------|----------|---------|
| 1 | 平民（Civilian） | ~94% | 無特殊身體強化，體能靠訓練提升，達到人類正常極限 |
| 2 | 武者（Warrior） | ~5% | 身體開始接受少量魔力浸潤，速度和力量達到常人2-3倍，能感知到有限的威脅 |
| 3 | 菁英武者（Elite） | ~0.8% | 魔力浸潤穩定，速度力量達到常人5-10倍，戰場上的核心戰力，「以一敵十」是真實水準 |
| 4 | 覺醒者（Awakened） | ~0.1% | 有一項或多項特殊能力覺醒（感知、體能強化超出常規），在戰場上能對抗數百人 |
| 5 | 少數強者（Apex） | < 0.01% | 能力已超出「量的積累」進入「質的不同」，能獨立改變中型戰役的走向 |

---

## 各等級詳細說明

### 平民（等級1）
- 佔絕大多數，普通士兵、商人、農民都在此列
- 訓練可以讓平民接近武者的體能，但不會觸發魔力浸潤
- 主角前世的第七邊境軍大多是平民中體能最好的那一批

### 武者（等級2）
- 魔力浸潤通常在15-25歲間自然觸發（持續高強度訓練是催化條件）
- 武者覺醒後，速度和力量的提升讓他們在近戰中對平民有壓倒性優勢
- 武者是邊境軍、私人護衛、城邦傭兵的主體
- 主角在12歲時覺醒（比平均提早4年）

### 菁英武者（等級3）
- 並非所有武者都能進入菁英層——需要特定的訓練方法和天賦
- 菁英武者是軍隊的指揮官層和貴族私人護衛的高端選項
- 主角在16歲時達到菁英水準
- 卡索在年輕時是菁英武者，後來覺醒

### 覺醒者（等級4）
- 覺醒通常由某個極端事件觸發（瀕死、強烈情感衝擊、長期極限訓練後的突破）
- 覺醒類型決定了能力方向：
  - 感知系：擴展感知範圍，能察覺微小動作和危險
  - 體能系：速度/力量再次躍升，達到常人20-30倍
  - 元素系（較少）：能控制基本元素（風、火等）
- 主角覺醒類型：感知系（18-19歲，被刺殺時觸發）

### 少數強者（等級5）
- 全大陸只有個位數
- 帝國最強的幾名將領在此列
- 黑鐵將雷蒙接近等級5但未完全達到（他是強覺醒者，接近頂端）
- 卡索是等級5——但他隱居，帝國不確定他的真實位置

## 力量與社會地位

| 力量等級 | 典型社會地位 |
|---------|------------|
| 平民 | 普通人；士兵通常在此列 |
| 武者 | 中等收入僱傭兵；邊境軍小隊長 |
| 菁英武者 | 上層僱傭兵；貴族私人護衛頭目；邊境軍指揮官 |
| 覺醒者 | 受各方勢力爭搶；有談判籌碼；可以「賣出好價錢」 |
| 少數強者 | 超出正常政治體系，自己決定站哪邊 |

**重要說明**：力量不等於地位——主角到達菁英武者時刻意隱藏，以避免被識別
